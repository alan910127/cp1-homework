> cpp standard: `c++17` <br>
> Compile with the option `-std=c++17` turned on.

- [c++11] Range-based for loop: `for (type var : iterable)`
    - e.g. `for (auto i : nums) { }`
- [c++17] Structured binding: `auto [identifier-list] = expression;`
    - e.g. `auto [x, y] = position;`
